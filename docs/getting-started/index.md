# Getting Started

Welcome to the Astro blog project! This section has everything you need to get the project running locally.

## Quick Start

### Prerequisites
- **Node.js**: 20+ (check with `node --version`)
- **npm**: 10+ (comes with Node.js)
- **Git**: For cloning and version control

### Setup (5 minutes)

```bash
# 1. Clone the repository
git clone https://github.com/kyleskrinak/astro-blog.git
cd astro-blog

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev

# 4. Open in browser
# Visit http://localhost:3000
```

## What's Happening?

The dev server watches your files and automatically reloads the browser when you make changes. Perfect for development!

## Common Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Start development server (http://localhost:3000) |
| `npm run build` | Build for production (creates `dist/` folder) |
| `npm run preview` | Preview production build locally |
| `npm run lint` | Check code for issues |
| `npm run format` | Auto-format code with Prettier |
| `npm run test:console` | Check for console errors |
